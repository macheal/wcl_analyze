# 任务：创建Boss选择器

## 任务标题
实现Boss下拉选择器组件

## 详细描述
创建一个Boss选择组件，根据报告ID获取并显示Boss列表：
- 调用 `/api/v2/code/get_wcl_boss_by_report_id` 接口获取Boss数据
- 使用Ant Design的Select组件实现下拉选择
- 支持搜索过滤功能
- 显示Boss名称、击杀状态等信息
- 处理加载状态和错误状态
- 将选中的Boss信息存储到状态管理

## 验收标准
- [ ] 正确调用API获取Boss列表

- [ ] 支持实时搜索过滤
- [ ] 加载状态显示正确
- [ ] 错误处理完善（网络错误、数据格式错误等）
- [ ] 选中的Boss信息正确存储
- [ ] 界面美观，用户体验良好
