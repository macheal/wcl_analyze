# 任务：创建Tab页导航

## 任务标题
实现Tab页切换导航组件

## 详细描述
创建Tab页导航组件，包含两个主要功能模块：
- 使用Ant Design的Tabs组件
- Tab 1: "命中技能查询" - 用于查看技能命中统计
- Tab 2: "卡雷什M7万相拳统计" - 用于查看特定Boss技能统计
- 实现Tab切换逻辑和状态管理
- 根据选中的Boss动态更新Tab内容
- 添加适当的图标和样式

## 验收标准
- [ ] Tab切换功能正常
- [ ] 两个Tab页标题正确显示
- [ ] Tab切换时保持选中状态
- [ ] 根据Boss选择动态更新内容
- [ ] 图标和样式美观
- [ ] 响应式设计，移动端适配良好
- [ ] Tab内容区域布局合理
