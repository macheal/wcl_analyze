# 任务：创建报告ID输入弹窗

## 任务标题
实现报告ID输入模态框组件

## 详细描述
创建一个模态框组件，用于在页面加载时获取用户输入的WCL报告ID：
- 使用Ant Design的Modal组件
- 实现输入框验证（非空验证、格式验证）
- 提供示例提示信息
- 实现确认和取消按钮功能
- 将输入的报告ID存储到全局状态或本地存储
- 错误处理和用户友好的提示信息

## 验收标准
- [ ] 页面加载时自动弹出模态框
- [ ] 输入框支持粘贴和手动输入
- [ ] 输入验证逻辑正确
- [ ] 确认按钮在验证通过后才可点击
- [ ] 取消按钮可以关闭弹窗（但会提示用户必须输入）
- [ ] 报告ID正确存储并可被其他组件访问
- [ ] 错误提示清晰易懂
