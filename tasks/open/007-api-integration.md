# 任务：API集成和错误处理

## 任务标题
实现API调用封装和统一错误处理

## 详细描述
创建统一的API调用层，包含以下功能：
- 封装axios实例，统一配置baseURL和请求头
- 实现请求拦截器，添加必要的认证信息
- 实现响应拦截器，统一处理错误响应
- 创建API服务模块，封装所有后端接口调用
- 实现重试机制和网络错误处理
- 添加请求loading状态管理
- 实现错误提示组件，统一显示错误信息

## 验收标准
- [ ] axios实例配置正确
- [ ] 请求和响应拦截器工作正常
- [ ] API服务模块封装完整
- [ ] 错误处理机制完善
- [ ] 网络错误和超时处理
- [ ] 错误提示用户友好
- [ ] loading状态管理正确
